# AIWD_LAB5
Artificial Intelligence in Web Development Laboratory 5

Description

This experiment demonstrates the implementation of an Event Countdown Timer using JavaScript. The timer calculates the remaining days, hours, minutes, and seconds until a specified future event date. The Date object is used to obtain the current time and the target event time. The difference between these times is calculated in milliseconds and converted into readable units.

The timer updates every second using setInterval() and dynamically updates the webpage using DOM manipulation methods. When the countdown reaches zero, the timer stops using clearInterval() and displays the message “Registration Closed”. This experiment illustrates JavaScript timing functions, DOM manipulation, and real-time UI updates.
